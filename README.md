这是一个基于回溯法的 N 皇后问题求解器，用于 AI 协作编程实践。

## 实现思路

使用**回溯法**递归地尝试在每一行放置一个皇后，并通过 `is_valid` 函数检查当前位置是否与已放置的皇后冲突（同一列、同一对角线）。如果找到一个合法的完整布局，则将其加入结果列表。
